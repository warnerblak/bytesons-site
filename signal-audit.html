A framework for how a project reads on first contact.

This tool evaluates signal — not hype, not price, not outcomes. It’s designed to reflect how your project reads to someone encountering it for the first time.

Framework (public)

The audit is built from a few consistent checks. You can use these manually, even without running the tool.

  • Center — what is the project in one sentence, without metaphors?
  • Mechanism — what is actually fixed, verifiable, or constrained?
  • Claims — what is promised vs. what is demonstrated?
  • Audience — who is it for, and what do they do next?
  • Trust surface — references, links, contract clarity, and obvious gaps.

If a project reads as “permanence” without specifying what’s fixed, it’s usually marketing. If it states one mechanism and one boundary, it’s usually real.
